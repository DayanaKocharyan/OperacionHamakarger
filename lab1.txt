Լաբորատոր աշխատանք 1
ԻԿՄ 2-րդ կուրս 202 խումբ
Քոչարյան Դայանա

1. Ստեղծել lab1 անունով դիրեկտորիա, որում ստեղծել 4 տեքստային ֆայլ՝ file.txt, file1.txt, file2.txt, file3.txt եւ 2 դիրեկտորիա՝ dir1, dir2։

cd desktop
mkdir lab1
cd lab1
touch file.txt file1.txt file2.txt file3.txt 
mkdir dir1 dir2

2. Կատարել հետեւյալ հրամանները եւ բացատրել տարբերությունը.
ls *.txt
ls * .txt

ls *.txt //կարտածի file.txt	file1.txt	file2.txt	file3.txt
ls * .txt //կարտածի No such file or directory

առաջինը պահանջում է թվարկել բոլոր ֆայլերը, որոնց անվանումը վերջանում է .txt սիմվոլներով, ուստի կտպի file.txt file1.txt file2.txt file3.txt, իսկ մյուսում բացատ կա, իսկ մենք չունենք նման ֆայլեր։ 

3. Ցուցադրել բոլոր տեքստային ֆայլերը (ավարտվում է .txt սիմվոլներով), որոնց
անվանումը սկսվում է f սիմվոլով եւ բաղկացած է 4 սիմվոլից։

ls f???.txt
//կարտածի file.txt

4. Ցուցադրել բոլոր տեքստային ֆայլերը, որոնց անվանումը սկսվում է file բառով, որին
հաջորդում է թվանշան։

ls file[0-9]*
//կարտածի file1.txt	file2.txt	file3.txt

5. Ցուցադրել բոլոր տեքստային ֆայլերը, որոնց անվան վերջին սիմվոլը փոքրատառ է,
կամ 1, 2 թվանշաններից որեւէ մեկը։

ls *[a-z12].txt
//կարտածի file.txt	file1.txt	file2.txt

6. Պատճենել /etc/passwd ֆայլը lab1 դիրեկտորիայում։

cp /etc/passwd lab1


7. Անվանափոխել պատճենված ֆայլը՝ այն դարձնելով new: Տեղափոխել new ֆայլը դեպի
dir1, այնուհետեւ տեղափոխել դեպի dir2:

mv lab1 new
mv new dir1
mv dir1/new dir2/

8. Անվանափոխել dir2-ը՝ այն դարձնելով dir3։ Տեղափոխել dir3-ը դեպի dir1։

mv dir2 dir3
lab1 % mv dir3 dir1/

9. Տեղափոխել dir3-ում գտնվող new ֆայլը դեպի lab1։

mv dir1/dir3/new ./

10. Ջնջել lab1 դիրեկտորիան։

cd ..
rm -r lab1
